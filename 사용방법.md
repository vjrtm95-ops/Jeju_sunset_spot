# 제주도 노을 맛집 - 사용 방법 가이드

## 1단계: 서버 실행하기

`서버실행.bat` 파일을 **더블클릭**하면 서버가 시작됩니다.

검은 창이 뜨면서 아래 메시지가 보이면 성공입니다:
```
Starting development server at http://127.0.0.1:8000/
```

그 다음 인터넷 브라우저(크롬, 엣지 등)를 열고 주소창에 입력:
```
http://127.0.0.1:8000
```

---

## 2단계: 처음 사용하기

1. **회원가입** 버튼 클릭 → 닉네임과 비밀번호 입력
2. 로그인하면 **제주도 지도**가 보입니다
3. 하단에 오늘 추천 명소 10곳이 카드 형태로 표시됩니다
4. 명소 카드 또는 지도 마커 클릭 → 상세 정보 확인
5. 상세 페이지에서 **후기 작성** 가능

---

## 3단계: 카카오맵 연결하기 (선택, 지도 보려면 필수)

> 지도 없이도 서비스는 작동합니다. 카카오맵은 무료로 사용 가능합니다.

1. https://developers.kakao.com 접속
2. 카카오 계정으로 로그인 (없으면 무료 가입)
3. **내 애플리케이션** → **애플리케이션 추가하기**
4. 앱 이름 입력 (예: 제주노을)
5. **앱 설정** → **플랫폼** → **Web** → 사이트 도메인에 `http://127.0.0.1:8000` 입력
6. **앱 설정** → **요약 정보** → **JavaScript 키** 복사

그 다음 `.env` 파일을 메모장으로 열어서:
```
KAKAO_MAP_KEY=복사한키를여기에붙여넣기
```

서버를 껐다가 다시 실행하면 지도가 표시됩니다!

---

## 4단계: 날씨 정보 연결하기 (선택)

> 날씨 없이도 서비스는 작동합니다. 무료 API입니다.

1. https://openweathermap.org/api 접속
2. **Get API key** 버튼으로 무료 계정 가입
3. API Keys 메뉴에서 키 복사
4. `.env` 파일에:
```
OPENWEATHER_API_KEY=복사한키를여기에붙여넣기
```

---

## 5단계: 관리자 페이지 사용하기

브라우저에서: http://127.0.0.1:8000/admin

- **관리자 아이디**: `admin`
- **관리자 비밀번호**: `admin1234`

관리자 페이지에서 할 수 있는 것:
- **노을 명소 관리**: 명소 추가/수정/삭제, 예시 사진 업로드
- **후기 관리**: 부적절한 후기 삭제 또는 숨기기

---

## Render 배포하기

1. https://github.com 에서 새 저장소(repository) 만들기
2. 이 폴더의 파일들을 업로드 (`.env` 파일은 업로드하지 마세요!)
3. https://render.com 가입 (무료)
4. **New +** → **Web Service** → GitHub 연결
5. 설정:
   - **Build Command**: `./build.sh`
   - **Start Command**: `gunicorn jeju_sunset.wsgi`
6. **Environment Variables** (환경 변수) 설정:
   - `SECRET_KEY` = 긴 랜덤 문자열 (https://djecrety.ir/ 에서 생성)
   - `DEBUG` = `False`
   - `KAKAO_MAP_KEY` = 카카오 키
   - `OPENWEATHER_API_KEY` = 날씨 키
   - `DATABASE_URL` = Render에서 PostgreSQL 무료 플랜 추가 후 자동 입력됨

## 자주 묻는 질문

**Q: 서버 껐다 켜는 방법?**
검은 창에서 `Ctrl + C` 누르면 꺼집니다. 다시 `서버실행.bat` 더블클릭.

**Q: 비밀번호 바꾸고 싶어요**
관리자 페이지(http://127.0.0.1:8000/admin) → Users → admin → Password 변경

**Q: 명소 사진을 추가하고 싶어요**
관리자 페이지 → Spots(노을 명소 목록) → 명소 선택 → 사진 추가

**Q: 지도에 새 명소를 추가하고 싶어요**
관리자 페이지 → 노을 명소 관리 → 명소 추가
- 위도/경도는 카카오맵에서 원하는 곳 우클릭 → "이 위치로" 선택하면 주소창에 좌표가 나옵니다
- 조망 방위각: 0=북, 90=동, 180=남, 270=서 (서쪽 해변이면 270 근처)
